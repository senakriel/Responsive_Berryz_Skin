“세나 스토리지”


이 홈페이지는 Berryz WebShare를 통해 구현된 개인 클라우드 공간입니다.

본 홈페이지의 스킨은 W3layouts의 Nakropol Web Template에 Wallel님의 반응형 v2.2 스킨의 소스코드의 도움을 받아 제작되었습니다.

“동영상 스트리밍”


Windows - 'ASX Streaming' 플러그인을 사용하면 해당 동영상을 확장자에 구애받지 않고 스트리밍으로 감상할 수 있습니다. mp4 파일의 경우 파일 우측에 표시되는 재생 버튼을 클릭하여, 세나 스토리지에서 제공하는 웹 플레이어를 통해 영상을 감상하실 수도 있습니다.(자막이 표시되지 않을 경우 하단의 cc버튼을 눌러 주세요.)

Android - MX Player와 같은 동영상 재생 어플리케이션을 사용하시면 플러그인을 사용하지 않고 동영상을 스트리밍으로 감상하실 수 있습니다.

iPhone - 기본 동영상 플레이어를 통하여 플러그인을 사용하지 않고 동영상을 스트리밍으로 감상하실 수 있습니다.(자막 미지원)

플러그인에서 Mp4 Player를 선택 시 같은 폴더 내에 있는 영상들을 동영상 우측에 리스트 형식으로 출력합니다. 옵션을 활용하여 개별 파일 아이콘 클릭 시에도 리스트를 출력하도록 할 수 있습니다.

“음악 플레이어”


Mp3 파일 우측에 표시되는 아이콘을 클릭하여 음악을 재생할 수 있습니다.

플러그인에서 Mp3 Player를 선택 시 같은 폴더 내에 있는 음악을 재생할 수 있는 플레이어를 사용하여 음악을 재생합니다. 옵션을 활용하여 아이콘 클릭 시 Mp3 Player를 열거나 Mp3 Player 사용 시 가사 지원 여부를 설정할 수 있습니다.

“이미지 뷰어”


이미지 파일 우측에 표시되는 아이콘을 클릭하여 Image Viewer를 사용하여 이미지를 감상할 수 있습니다. 상단의 전체보기/한장보기를 활용하여 같은 폴더 내의 이미지를 모두 표시/한 장만 표시 할 수 있습니다.

옵션을 활용하여 플러그인에서 Image Player를 선택 시 같은 폴더 내에 있는 이미지를 모두 출력하는 옵션을 활용할 수 있습니다.

“사용자 지정 테마”


옵션을 통해 사용자 지정 테마를 활용하여 홈페이지를 이용하실 수 있습니다. 현재 사용자 테마 색상은 입니다.
